How to run the program: 

1. Clone or download the repository to your local machine 

2. Navigate to the project folder (../Event-Management-System)

3. Navigate to the front-end directory (run terminal command: cd ../Event-Management-System/client)

4. Build the React App (npm dev build)

5. Navigate to the back-end directory (run terminal command: cd ../Event-Management-System/server)  

6. Install dependencies (run terminal command: npm install) 

7. Start back-end server (run terminal command: npm start)
